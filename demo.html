Try Nib

A plain page with standard text fields, so you can see Nib work in seconds.

How to test

  1. Install / enable the Nib extension. No login or setup is required.
  2. In the text area below, select the pre-filled sentence (triple-click, or drag across it).
  3. A small Refine button appears next to your selection — click it.
  4. A popup shows a cleaner version. Click Replace to swap it into the field (only the selected text changes), or Copy / Close.

Select the text above, then click the Refine button that appears.

Nib also works in standard single-line input fields like this one.

Notes for reviewers

Nib activates only when you select text inside a standard <input> or <textarea> field and is otherwise invisible. Only the text you select is sent to our endpoint; nothing is stored. See the privacy policy.
